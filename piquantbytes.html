Piquant Bytes

Piquant Bytes is a Recipe App for home-made, tasty food recipes.

Piquant Bytes is a Responsive Web app for Recipes that aims to help users in their quest for home-made, tasty food. This web app tries to address the pain points that users face with other recipe websites and improve upon them. The app is responsive and has breakpoints for mobile, tablet, laptop and desktop screen sizes.
